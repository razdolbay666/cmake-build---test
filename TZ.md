# Плагин для VSCode - техническое задание

## Требуемый функционал 

* Регистрация команды cmake-build-test.runBuildTest в системе команд VSCode
* Автоматическое определение рабочей области с проверкой наличия открытой workspace
* Создание и управление терминалом для выполнения команд сборки
* Последовательное выполнение команд CMake:
  * cmake -B build - конфигурация проекта
  * cmake --build build - сборка проекта
  * ctest -V --test-dir build - запуск тестов с детализированным выводом

## Пользовательский интерфейс 

* Статус-бар кнопка с иконкой "play" и текстом "Build & Test"
* Контекстный показ кнопки - отображается только при наличии файла CMakeLists.txt в проекте
* Tooltip с описанием функциональности "Run CMake Build and Test"
* Информационные сообщения о начале процесса сборки

## Зависимости 

* Работа с VSCode версии 1.105.1 или выше
* СМаke версии 3.10 или выше
* Компилятор: GCC, Clang, MSVC или любой другой совместимый.
